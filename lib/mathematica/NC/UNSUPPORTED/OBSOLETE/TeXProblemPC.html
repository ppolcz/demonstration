Problems with LaTeX with a PC

  1. LaTeX is not needed to run NCAlgebra and is not needed to run most of NCGB.
  2. The problem faced with NCGB using LaTeX is almost always with NCGB finding it automatically.
  3. If you do not have LaTeX installed properly on your PC, then talk to your system administrator or obtain a distribution of LaTeX for the PC from the internet. MikTeX (http://www.miktex.org/) is a good choice.
  4. You possibly have to modify the file NCGBTeXSetUp.m in the NC/NCGB/MmaSource/ directory. Look at the file to see what to do.
