Suppose that you want to unzip a file Win.zip. Choose the (one) correct step of the few which are listed here.
  1. Download your favorite program to unzip a file. You can obtain a free evaluation copy of winzip (from http://www.winzip.com) or winrar (from http://www.rarlab.com/download.htm) Then use this program to unzip the file TheFile.zip.
OR
  1. If you are using Windows XP:
    1. Find the file with Internet Explorer (starting with "My Computer" if necessary) and click on the zip file.
    2. You should see an NC directory. Right-click on the NC directory and choose the "copy" option.
    3. Click on the "up directory" icon in the toolbar.
    4. Choose to paste menu option from the "Edit" menu of Internet Explorer.
